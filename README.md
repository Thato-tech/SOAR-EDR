# SOAR-EDR

## Objective


The purpose of this project is to enhance current security operations through combining SOAR automation with EDR for the purpose of improving alert processing time, increasing speed in responding to incidents and enhancing overall threat identification. This project will automate many of the repetitive tasks currently performed manually, increase end-point visibility, create a standardized process for responding to threats, and provide scalability in responding to future potential threats.

### Skills Learned


-Ability to use automation to enable the SOC to complete multiple redundant security tasks so that analysts can be more efficient and have a lighter workload
.
-Ability to integrate SOAR workflow automation into EDR tools to allow for better coordination in detecting and responding to threats.

-Ability to analyze endpoint alerts and behavioral data to find where there may be malicious activity occurring on your systems.

-Ability to create and design rule-based playbooks that provide consistent and scalable incident response.

-Better understanding of how threat intelligence is enriched and automatically used as part of an automated decision making process.

-Ability to isolate any compromised endpoints and contain threats by utilizing your EDR tool's control capabilities.
